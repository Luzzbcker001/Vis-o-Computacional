# Vis-o-Computacional
Um projeto simples de visão computacional, onde o seu mouse acompanha o seu dedo indicador
Para clicar, junto seu dedo polegar com o médio, fazendo uma pinça
Todos os prints estão comentados dentro do código, abilite se quiser.

print(f"Dedo {id}: {cx}, {cy}") -> Mostra a coordenada de cada parte do dedo
ID 4 - Polegar
ID 8 - Indicador
ID 12 - Médio
ID 16 - Anelar
ID 20 - Mínimo

print(f"Distância: {dist}") -> Mostra a distância entre o polegar e o dedo médio

print("Clicked") -> Mostra quando o clique for executado

Libs usadas (algumas nativas do python):
opencv-python
mediapipe
pyautogui
math (nativa do python)
